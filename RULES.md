# RULES.md — 망각의 지뢰 게임 규칙

이 문서는 게임 규칙의 **단일 출처(Single Source of Truth)** 입니다. 규칙 변경은 여기에 먼저 반영하고, 코드(`src/game/`) 와 다른 문서(`PLAN.md`, `CLAUDE.md`) 는 이 파일을 참조합니다.

## 게임 개요
- 플레이 방식: 로컬 2인 **핫시트** (한 화면을 번갈아 씀, 서버 없음)
- 보드: **11×11** (좌표 표기: `a`~`k` × `1`~`11`)
- 플레이어별 지뢰: **15개**
- 시작 위치: **P1 = a11, P2 = k1**
- 보물 위치: **a1, f6(중앙), k11**

## 지뢰 배치 (SETUP 단계)

### 배치 금지 구역
- 각 시작점 주변 **2×2** 영역 (체비쇼프 거리 ≤ 1)
  - P1(a11) 주변: **a10, a11, b10, b11**
  - P2(k1) 주변: **j1, j2, k1, k2**
- 보물 칸: **a1, f6, k11**

### 기타 배치 제약
- 같은 플레이어는 한 칸에 지뢰 **1개**까지 (상대와 겹치는 것은 허용)
- 배치 제한 시간: **10분**. 만료 시 남은 지뢰를 허용 구역에서 랜덤 배치 후 자동 제출.
- 배치 중에는 **자기 지뢰**가 보이지만, 배치 완료 후(HANDOFF 이후) 모든 지뢰는 **숨김** — 자기 것 포함. 암기 게임이 되는 핵심 규칙.

## 이동 규칙
- 현재 말 기준 **8방 인접 칸**으로만 이동
- **상대 말이 있는 칸**은 이동 불가
- 보드 밖 이동 불가
- 셀을 클릭하면 "이동 선택" 상태가 되고, 실제 이동은 **예 / 아니오** 확인 버튼으로 확정. 실수 클릭 방어용.

## 이동 결과 평가 (순서대로 적용)
아래 순서대로 검사하며, 처음 해당하는 규칙만 적용됩니다.

1. **대상이 보물 칸**이면 → 수집 순서에 따라 **+10 / +15 / +20점**. 세 개 모두 수집되는 순간 게임 종료.
2. **대상에 지뢰가 1개 이상** (양 플레이어 합계) 있으면 → **-5점**, 그 칸의 지뢰 전부 제거, **강제 이동** 단계 진입.
3. **대상이 빈 칸**이면
   - 첫 방문 → 주변 8칸의 **지뢰 개수 (중복 포함)** 만큼 득점
   - 이미 점수 획득된 칸 (재방문) → 점수 없음

> 📝 지뢰를 밟은 칸은 "점수 획득 기록"에 추가되지 않으므로, 이후 누군가 다시 방문하면 "첫 방문 빈 칸"으로 평가되어 주변 지뢰 수만큼 득점 가능합니다.

## 강제 이동
지뢰를 밟은 직후, 해당 플레이어는 자동으로 강제 이동 단계에 들어갑니다.

- **자기 시작점 기준 8방 인접 칸** 중 한 칸을 선택해 이동
- 조건: 보드 내, 상대 말이 없는 칸만 후보 (시작점이 코너라 최대 3칸)
- 일반 이동과 동일하게 **예 / 아니오** 확인 절차 적용

## 공개 / 비공개 정보
- ✅ **공개**: 현재 지뢰가 설치된 칸 수 (양 플레이어 지뢰 Set 의 합집합 크기). 지뢰가 밟혀 제거될 때마다 함께 감소.
- ✅ **공개**: 수집된 보물 개수 (0 / 1 / 2 / 3)
- ✅ **공개**: 각 플레이어의 현재 점수
- ❌ **비공개**: 개별 지뢰 위치 (배치 완료 후에는 자기 지뢰도 포함)
- ❌ **비공개**: 방문 이력과 이전 이동에서 얻은 점수 숫자 — 보드에 아무런 시각 표시도 남기지 않음. (기억력이 핵심이라 UI 에 힌트를 남기면 게임이 성립하지 않음.)

## 점수 요약

| 사건 | 점수 |
| --- | --- |
| 첫 번째 보물 | **+10** |
| 두 번째 보물 | **+15** |
| 세 번째 보물 | **+20** |
| 지뢰 밟음 | **-5** |
| 빈 칸 첫 방문 | 주변 8칸의 지뢰 수 (중복 포함) |
| 빈 칸 재방문 | 0 |

## 승리 조건
- 세 번째 보물이 수집되는 즉시 게임 종료 (`ENDED` phase).
- 총점이 **높은** 플레이어가 승리.
